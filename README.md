# PawnChess
A game of chess with only pawns available

## Our Team
- Adityo Anggraito

## Prerequisites
Java 8

## How to Run
Option 1: Open cmd, run 'java PawnChessGUI'

Option 2: Double-click PawnChess.jar

Choose the difficulty, and you're good to go :)

## Future Update
